## [level 2] 방문 길이

### 문제 접근 방식 --> 다시 풀어보자~~~~
- 방향을 담는 배열을 map에 담아서 방향대로 좌표를 이동시켜 값을 구함
- 이미 도달한 좌표였다면 이미 지나왔던 좌표에서 이동한 것인지 비교를 한다. --> 잘못된 접근 방식
- 단방향으로만 어떻게 계산할지 생각했다 --> 양방향 둘다 고려해야 함
