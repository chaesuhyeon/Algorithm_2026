## [level 1] 두 개 뽑아서 더하기
[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/68644)

### 문제 접근 방식
- 배열의 길이가 정해져있지 않아서, 처음에는 ArrayList로 선언했고, steam의 distict()메서드를 사용하여 중복을 제거하고, ArrayList를 int[]로 바꿔줘서 풀었음
- 서로 다른 인덱스의 합을 구해야하기 때문에 이중 반복문을 사용하여 구현 했음

** 참고 **  
HashSet이라는 자료구조를 사용하면 distinct()를 사용하지 않고도 중복을 제거할 수 있음 -> 코드 1줄 삭제 가능

### 성능 요약
- 실행 시간: 평균 약 4.3ms (최소 2.32ms / 최대 7.51ms)
- 메모리 사용: 평균 약 82MB (70.1MB ~ 88.7MB)

### 시간 복잡도 분석
N은 numbers의 길이이며
- 이중 반복문을 통해 두 수의 합을 구하는 연산의 시간 복잡도: O(N^2)
- Set으로 만들때의 시간 복잡도 O(N^2)
- N^2개의 데이터를 정렬하는데는 O(N^2 log(N^2))

=> 최종 시간 복잡도는 O(N^2 log(N^2))
