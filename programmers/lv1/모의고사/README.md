## [level 1] 모의고사
***
[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/42840)

### 문제 접근 방식
- 모든 경우를 다 따져서 max값을 알아낸 후에 max값과 같은 수포자를 찾도록 조건문을 사용하여 구현했음
- 처음에는 수포자가 답찍는 패턴만 가지고 했는데 패턴이 반복된다는 부분을 놓쳐서 런타임 에러가 발생했음
- 반복되는 인덱스값을 제대로 구해서 수정하니 통과하였음

### 성능 요약
- 실행 시간: 평균 약 4.6ms (최소 2.48ms / 최대 17.12ms)
- 메모리 사용: 평균 약 86.1MB (최소 72.6MB / 최대 98.4MB)
